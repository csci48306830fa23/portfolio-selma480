Selma Spahic's VR Portfolio

Demo 1

In this demo, I created a spawner that takes keyboard input to spawn certain GameObjects. Concepts here: prefabs, keyboard input, GameObject self destruction.

Demo 2

This demo demonstrates the OnCollisionEnter and how an action can be done once a collion has been detected.

Demo 3

For this demo, I created a cannon which shoots cannonballs (based on mouse input) to demonstrate how the behavior of them will change depending on the physic material. I also demonstrate rigid bodies.

Demo 4

This demo demonstrates Unity UI using a slider, dropdown menu, and a button. These UI's change some aspect of the cube.

Demo 5

This demo uses Oculus Integration for hand tracking and interacting with objects on the scene.

Demo 6

This demo uses Oculus Integration and shows the ability to interact with and throw objects from the scene.

Demo 7

This demo uses XR Interaction Toolkit to create a shooting simulation. The gun can be picked up with the controller and shoots when the trigger is pressed. Targets explode when hit.

Demo 8

For this demo, I explored with the Mixamo avatars and using Animator layers to combine animations.
